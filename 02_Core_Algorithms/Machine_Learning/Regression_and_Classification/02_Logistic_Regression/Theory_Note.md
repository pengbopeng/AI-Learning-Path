# 逻辑回归 (Logistic Regression)

## 1. 核心概念
虽然叫“回归”，但它是**处理二分类问题**（0 或 1）的线性模型。
*   **直觉**：在线性回归的直线上，套一个 Sigmoid 函数，把值压缩到 [0, 1] 之间。
*   **例子**：垃圾邮件检测、疾病诊断、金融风控。

## 2. 关键公式
### Sigmoid 函数 (激活函数)
$$ g(z) = \frac{1}{1 + e^{-z}} $$
*   作用：把 $(-\infty, +\infty)$ 映射到 $(0, 1)$。

### 预测公式
$$ \hat{y} = g(wx + b) $$
*   若 $\hat{y} > 0.5$：预测为 1 (Positive)
*   若 $\hat{y} \leq 0.5$：预测为 0 (Negative)

## 3. 损失函数 (Loss Function) - ⭐面试必考
逻辑回归**不使用**线性回归的 MSE（均方误差），因为 Sigmoid 导致 MSE 是“非凸”的（有很多局部最低点，梯度下降会卡住）。

我们使用 **对数损失 (Log Loss)**，也叫**二元交叉熵 (Binary Cross Entropy)**：
$$ J(w,b) = - \frac{1}{m} \sum [y \log(\hat{y}) + (1-y) \log(1-\hat{y})] $$

*   如果真实值 $y=1$，我们希望 $\hat{y}$ 越接近 1 越好（loss 接近 0）。
*   如果真实值 $y=0$，我们希望 $\hat{y}$ 越接近 0 越好。

## 4. 优缺点
*   **优点**：计算简单，可解释性强（能看到各个特征的权重，知道哪个特征影响大）。
*   **缺点**：只能处理线性可分的数据（分界线是一条直线），容易欠拟合。
