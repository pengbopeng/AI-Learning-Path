# 🧠 AI-Learning-Path (我的AI进阶之路)

本项目基于**六大阶段**思维导图构建，记录了我从零开始学习人工智能的全过程。

## 📂 目录结构
- **01_AI_Essentials**: Python基础、数学基础 (Numpy/Pandas/微积分/线性代数)
- **02_Core_Algorithms**: 机器学习(手撕算法)、深度学习(CNN/RNN/Transformer)
- **03_Computer_Vision**: OpenCV、YOLO系列、图像分割
- **04_NLP**: BERT、知识图谱、Huggingface实战
- **05_Advanced_Deployment**: 大模型Agent、强化学习、模型部署
- **06_Special_Updates**: 论文复现、科研实战、行业案例
